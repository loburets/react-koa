# React + Koa - sign up and sign in

## Requirements:
- Node v10.x
- NPM v5.x
- MysQL

## How to install requirements:
- npm install
- cd ./frontend && npm install
- cd ../backend && ../node_modules/.bin/sequelize db:migrate
- set DB params in backend/config/config.json 

## How to run app:
1. npm start
2. open http://localhost:3000/
